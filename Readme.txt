AvaClo (あばくろ) - VRChat Avatar Manager v1.0.0
================================================================

この度は「AvaClo」をダウンロードいただきありがとうございます。
本ソフトはVRChatのアバターを管理・着替えやすくするための非公式ツールです。

■ 概要
沢山のアバターをカテゴリー分け（素体別）したり、お気に入りフォルダで整理したり、
タグ付けや検索機能を使って素早く目的のアバターを見つけることができます。
インストール不要のポータブル版ですので、USBメモリ等に入れて持ち運ぶことも可能です。

■ 使い方
1. zipファイルを解凍してください。
2. フォルダ内の「AvaClo(あばくろ).exe」をダブルクリックして起動します。
3. VRChatのアカウントでログインしてください。
   (2段階認証メール/アプリにも対応しています)

■ データについて
設定やログイン情報は、exeファイルと同じフォルダに以下のファイルとして保存されます。
- settings.json (設定・タグ・カテゴリ等)
- sessions.json (ログインセッション)

フォルダごと移動すれば、設定もそのまま引き継がれます。
(削除したい場合はフォルダごと削除してください)
